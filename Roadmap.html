Step-by-Step Build Guide

1

Prepare Your Workspace

Set up a clean, static-free environment for your PC build.

Clean workspace setup

💡 Pro Tips:

  • Use a large, flat surface like a dining table
  • Remove carpets or wear anti-static wrist strap
  • Keep component boxes for organization
  • Have good lighting or use a desk lamp
2

Unpack & Inventory Components

Carefully unpack all components and verify you have everything needed.

Components laid out and organized

📋 Checklist:

  • PC Cabinet with all screws and standoffs
  • Motherboard with I/O shield
  • Processor with stock cooler (if included)
  • Graphics card
  • RAM modules
  • SSD with SATA cables
  • Power supply unit (PSU)
3

Install I/O Shield

Install the motherboard's I/O shield into the rear panel of the case.

I/O shield installed in case

⚠️ Important:

  • Install from inside the case, pressing outward
  • Ensure all corners are properly seated
  • Check that metal tabs don't block ports
  • This is easier to do before motherboard installation
4

Install Motherboard Standoffs

Install brass standoffs in the case to elevate the motherboard and prevent short circuits.

Standoffs screwed into case

🔧 Installation Tips:

  • Only install standoffs where motherboard has mounting holes
  • Hand-tighten first, then use screwdriver carefully
  • Check motherboard manual for exact placement
  • Extra standoffs can cause short circuits
5

Install Processor (CPU)

Carefully install the processor into the motherboard socket.

CPU installed in motherboard socket

🚨 Critical Steps:

  • Open CPU socket by lifting retention arm
  • Align CPU with socket (look for alignment markers)
  • CPU should drop in without force
  • Lower retention arm to secure CPU
  • Never force or bend CPU pins
6

Install RAM Memory

Insert RAM modules into the appropriate slots on the motherboard.

RAM modules installed in motherboard

📍 Installation Guide:

  • Open RAM slot clips
  • Align RAM notch with slot notch
  • Press down firmly until clips snap into place
  • For dual-channel, use slots 2 and 4 first
  • Check motherboard manual for optimal slot configuration
7

Install M.2 SSD

Install the SSD into the M.2 slot on the motherboard.

M.2 SSD installed on motherboard

💽 SSD Installation:

  • Locate M.2 slot on motherboard
  • Insert SSD at 30-degree angle
  • Press down and secure with provided screw
  • Remove any heatsink stickers if present
  • Check motherboard manual for multiple M.2 slots
8

Install Motherboard in Case

Carefully place the motherboard into the case and secure with screws.

Motherboard mounted in PC case

🔩 Mounting Process:

  • Align motherboard with I/O shield
  • Line up mounting holes with standoffs
  • Start with corners, then center screws
  • Don't overtighten - snug is enough
  • Ensure no cables are pinched underneath
9

Install Power Supply (PSU)

Mount the power supply unit in the designated area of the case.

PSU installed in case

⚡ PSU Installation:

  • Fan should face down if case has bottom ventilation
  • Otherwise, fan faces up into the case
  • Secure with four screws from outside the case
  • Ensure PSU switch is in OFF position
  • Leave power cable disconnected for now
10

Install Graphics Card

Insert the graphics card into the PCIe x16 slot on the motherboard.

Graphics card installed in PCIe slot

🎮 GPU Installation:

  • Remove PCIe slot covers from case
  • Open PCIe slot clip
  • Align card with top PCIe x16 slot
  • Press down firmly until it clicks
  • Secure with screws to case bracket
11

Connect Power Cables

Connect all necessary power cables from the PSU to motherboard and components.

Power cables connected to motherboard

🔌 Power Connections:

  • 24-pin main power to motherboard
  • 8-pin CPU power (top-left of motherboard)
  • PCIe power cables to graphics card
  • SATA power to SSD (if not M.2)
  • All connectors have unique shapes - don't force
12

Connect Front Panel Connectors

Connect the case's front panel connectors to the motherboard.

Front panel connectors attached

🔗 Front Panel Connections:

  • Power button and reset button
  • Power LED and HDD LED
  • USB headers from case
  • Audio header (HD Audio)
  • Check motherboard manual for exact pin layout
13

Install CPU Cooler

Install the CPU cooler to keep your processor at safe temperatures.

CPU cooler mounted on processor

❄️ Cooling Installation:

  • Apply thermal paste if not pre-applied
  • Mount cooler according to manufacturer instructions
  • Connect cooler power cable to CPU_FAN header
  • Ensure even pressure across CPU surface
  • Check clearance with RAM modules
14

Cable Management

Organize and secure all cables for better airflow and aesthetics.

Clean cable management setup

🗂️ Cable Management:

  • Route cables behind motherboard tray
  • Use cable ties or velcro straps
  • Keep cables away from fans
  • Group similar cables together
  • Leave some slack for future upgrades
15

Final Checks

Perform final inspection before powering on your new PC.

Final PC build inspection

✅ Pre-Power Checklist:

  • All power connections secure
  • RAM properly seated and clicked in
  • Graphics card fully inserted
  • CPU cooler connected and secured
  • No loose screws in case
  • PSU switch still in OFF position
16

First Boot

Power on your PC for the first time and check for successful boot.

PC powered on and booting

🚀 First Boot Process:

  • Connect monitor to graphics card (not motherboard)
  • Flip PSU switch to ON
  • Press case power button
  • Check for fans spinning and lights
  • Look for BIOS/UEFI screen on monitor

Common Issues & Solutions

PC Won't Turn On

No lights, fans, or activity when pressing power button.

Solution: Check PSU switch, reseat power cables, verify front panel connectors, and ensure RAM is properly seated.

No Display Output

PC turns on but monitor shows no signal.

Solution: Connect monitor to graphics card (not motherboard), reseat GPU and RAM, check all power connections.

Boot Loop

PC turns on, then off, then on again repeatedly.

Solution: Clear CMOS, check RAM compatibility, verify CPU installation, ensure adequate power supply.

High Temperatures

CPU or GPU running too hot during operation.

Solution: Check thermal paste application, verify cooler mounting, ensure case fans are working, improve airflow.
